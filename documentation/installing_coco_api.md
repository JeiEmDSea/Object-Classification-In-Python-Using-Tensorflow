# Installing COCO API (pycocotools)

1. Install **git**:
```
conda install git
```

2. Install **pycocotools** by typing:
```
pip install "git+https://github.com/philferriere/cocoapi.git#egg=pycocotools&subdirectory=PythonAPI"
```

3. Keep the command window open.
